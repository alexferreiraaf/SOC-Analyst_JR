# **Exercício 1 — Explicação simples**
Resposta a Incidentes é o processo que uma equipe usa para lidar com problemas de segurança.
Ela serve para identificar ataques, parar o problema e reduzir danos.
Depois, o time corrige a falha que permitiu o ataque.
O sistema é colocado de volta em funcionamento com segurança.
Por fim, o ocorrido é analisado para evitar que aconteça novamente.
