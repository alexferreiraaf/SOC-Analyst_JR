# Dia 68 — Dashboard SOC: Detecção de Brute Force

## Nome do Dashboard
SOC - Security Monitoring

## Objetivo
Fornecer visão operacional para o SOC identificar rapidamente ataques de força bruta.

## Painéis incluídos
1. Failed Logins Over Time (Line Chart)
2. Top Attacking IPs (Bar Chart)
3. Targeted Users (Table)
4. Total Failed Logins (Single Value)

## Filtros Globais
- Time Picker (Last 24 hours)

## Público-alvo
Analistas SOC N1 / N2

## Uso em SOC
- Monitoramento contínuo
- Detecção rápida de picos de ataque
- Priorização de incidentes
