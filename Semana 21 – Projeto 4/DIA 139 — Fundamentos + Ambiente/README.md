# Projeto 4 - Investigação de Phishing

## Objetivo

Investigar e-mails suspeitos utilizando análise de logs, Splunk, Regex e Python.

## Ferramentas

* Splunk
* Python 3
* Regex
* CSV

## Estrutura

semana21_phishing_project/

├── logs/

├── splunk_queries/

├── script/

├── report/

└── README.md

## Cenário

Foram analisados logs de e-mail contendo URLs potencialmente maliciosas para identificar campanhas de phishing.

## Resultados Esperados

* Identificação de domínios suspeitos
* Extração automática de URLs
* Classificação de risco
* Relatório final em CSV
