# Dia 79 — Comandos Bash Essenciais para SOC

## Comandos utilizados

### grep
Usado para filtrar eventos suspeitos, como falhas de login.

### awk
Extrai campos específicos dos logs, como IPs e usuários.

### sort | uniq -c
Conta ocorrências e identifica padrões repetidos.

### wc -l
Conta o volume total de eventos.

Esses comandos permitem investigação SOC mesmo sem SIEM.
